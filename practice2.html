PRACTICE - SET

☢ Write a program add Natural numbers :

Program: 
#include <stdio.h>
int main()
{
int n,k=0;
scanf("%d",&n);
for (int i = 1; i <= n; i++)
{
    k = k+i;
}
printf("Sum of %d Natural number is = %d",n,k);
return 0 ;
}
OUTPUT: 
100
Sum of 100 Natural number is = 5050

☢ Write a program find factorial of given number :

Program: 
#include <stdio.h>
int main()
{
double n,k=1;
scanf("%lf",&n);
for (int i = 1; i <= n; i++)
{
    k = k * i;
}
printf("Factorial of %lf is = %lf",n,k);
return 0 ;
}
OUTPUT:

case 1

15
Factorial of 15.000000 is = 1307674368000.000000

case 2:

50
Factorial of 50.000000 is = 30414093201713376000000000000000000000000000000000000000000000000.000000
(Factorial of 50.000000 is = 3.04141e+064)
